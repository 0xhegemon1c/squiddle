Irongeek has a collection of hacking and doxing tutorials, it is run by Adrien Crenshaw, a graduate of Indiana Universty.
Codecademy is where I had my first introduction to coding before taking a few college classes. Despite what some might say it is an excellent resource for beginners, and it's free.
Nullbyte is a site rife with hacking tutorials, the "greenhorn" hacker will find a lot of useful tutorials here.
Build-electronic-circuits contains learning resources, a helpful introduction, and tutorials for building your own electronics.
Inteltechniques contains OSINT tools, it is run by Michael Bazzell, who spent 18 years as a government computer crime investigator.
During the majority of that time, he was assigned to the FBI's Cyber Crimes Task Force.
Hackthissite and thisislegal are both sites containing "hackng wargames" and practical hands-on hacking challenges.
Vedbex is a useful and fun site containing a web-stressor, skype, DNS, Email and phone resolvers, bots, and IP loggers, leaked databases and much more.
Z-shadow is a site which hosts and provides phishing pages, just create a fake account and give a victim the phishing page and Z-shadow will alert you when credentials have been stolen.
Leakedsource contains a plethora of leaked databases, and results with passwords in plain text as well as other useful credentials, for a small fraction of a bitcoin. Prices vary but are usually very low.
Goo.gl is a URL shortener, I chose this one because individuals are more likely to trust what looks like a google address versus a "bitly".
Codepen is a site that lets one freely host their code for other individuals to look at and alter.
Learngit branching is a useful site to learn Git, is a version control system for tracking changes in computer files and coordinating work on those files among multiple people.
VIM is a useful text editor program for Unix. It was built to make creating and changing any kind of text very efficient. this site teaches you VIM in the form of a game.
FreeCodeCamp is a site that teaches one how to code, much like Codecademy but more intensive!
Hak5 is your one-stop shop for hacking gear such as WiFi Pineapples, USB Rubber Duckies, Lan Turtles, etc (I wish they carried USB killers).
